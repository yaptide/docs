## What is YAPTIDE?

**YAPTIDE** (Yet Another Particle Transport IDE) is an open-source web application for Monte Carlo particle transport simulations. It supports **SHIELD-HIT12A**, **FLUKA**, and **Geant4** — three of the most widely used codes in radiation physics.

You design 3D geometry in a browser-based editor, configure a particle beam and scoring detectors, and either generate simulator input files or run simulations directly — all without leaving the page.

## How YAPTIDE Works

YAPTIDE has three execution modes. You don't need the full stack for every one.

| Mode                | What you can do                                            | What you need                          |
| ------------------- | ---------------------------------------------------------- | -------------------------------------- |
| **Demo**            | Edit geometry, generate input files, run Geant4 in-browser | Just a browser                         |
| **Celery (direct)** | Submit SHIELD-HIT12A / FLUKA jobs to local workers         | Docker stack or local backend + Redis  |
| **Batch (SLURM)**   | Submit to HPC clusters (e.g. PLGrid Ares)                  | Full stack + Keycloak + cluster access |
