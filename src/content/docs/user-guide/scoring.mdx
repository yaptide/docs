Scoring tells the simulator **what to measure** and **where to measure it**.
You need to define several items:

- **Detector** — the scoring geometry (where to score)
- **Quantity** — what physical quantity to score (dose, fluence, etc.)
- **Output** — a group of quantities scored in the same detector
- **Filter** (optional) — restrict scoring to specific particles (e.g. "protons only")
- **Differential modifiers** (optional) — score in energy or angle bins

## Detectors

Detectors can be added by selecting **Object > Detector** in the menu bar, or clicking **+ DETECTOR** in the DETECTORS section of the GEOMETRY tab.

![Adding a detector from menu bar](/docs/assets/scoring/menu_detector_add.png)

![Adding a detector from DETECTORS section](/docs/assets/scoring/detector_add.png)

### Narrow cylinder along the beam axis

<Tabs>
<TabItem label="SHIELD-HIT12A">

Add a cylinder with radius 1 cm and length 12 cm along the beam axis, spanning from −2 cm to 10 cm in Z.
Press **+ DETECTOR** and configure:

- Rename to `AlongBeamAxis`
- Set position (centre) to `(0, 0, 4)` cm
- Select geometry type: **Cyl** (cylinder)
- Under GRID, set 120 bins along the Z axis (each bin = 1 mm)

## Scoring filters

Often, scoring a quantity for all particles isn't very useful. We want to score proton fluence only.

<Tabs>
<TabItem label="SHIELD-HIT12A">

Define a filter in the FILTERS section of the SCORING tab.
Press **+ FILTER** (or **+ CUSTOM FILTER** in SHIELD-HIT12A). Name it `Protons`.

Add two rules:
1. Click **ADD RULE**, select `Z = 1`

![Rule for Z](/docs/assets/scoring/protons_rule_z_eq_1.png)

2. Add another rule: `A = 1`

![Rule for A](/docs/assets/scoring/protons_rule_a_eq_1.png)

Apply the filter by checking the **Filter** checkbox in the `Fluence` quantity under `YZSlab`, then select `Protons`.

![Apply filter](/docs/assets/scoring/apply_filter.png)

:::tip
For SHIELD-HIT12A, you can use **+ PARTICLE FILTER** and select "Protons" directly for the same effect.
:::

</TabItem>
<TabItem label="FLUKA">

:::note[Under development]
FLUKA filter configuration details will be added once support is complete.
:::

</TabItem>
<TabItem label="Geant4">

Define a filter in the FILTERS section. Press **+ FILTER**, name it `Protons`, set type to **particle**, and select **Proton**.

![Rule for protons](/docs/assets/scoring/geant4_filter_protons.png)

Apply the filter by checking the **Filter** checkbox in the `Fluence` quantity, then select `Protons`.

![Apply filter](/docs/assets/scoring/geant4_apply_filter.png)

</TabItem>
</Tabs>
