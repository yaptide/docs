## Full version (PLGrid)

The full version is hosted on PLGrid infrastructure at:

**[yaptide.c3.plgrid.pl](https://yaptide.c3.plgrid.pl)**

With a PLGrid account you can:

- Run **SHIELD-HIT12A** and **FLUKA** simulations on cloud or HPC resources
- Submit **direct runs** (fast, limited to ~15 cores in C3 PLGrid Cloud)
- Submit **batch runs** on the Ares supercomputer (SLURM, up to thousands of cores)
- Store and retrieve simulation results

![Simulations page — available for logged-in users](/docs/assets/simulations.png)

:::note
Access requires a PLGrid account with the `PLG_YAPTIDE_ACCESS` service enabled. Contact the administrator if you need access.
:::
