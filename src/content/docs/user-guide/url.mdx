## Loading a project from a URL

You can load a YAPTIDE project file directly by appending its URL as a query parameter:

```
https://<editor_url>?<project_file_url>
```

For example:

```
https://yaptide.github.io/web_dev/?https://raw.githubusercontent.com/yaptide/ui/master/src/ThreeEditor/examples/ex1.json
```

This is useful for sharing simulation setups with colleagues — just send them the link.
Navigate to the **Results** tab in the main menu to view pre-computed results.
